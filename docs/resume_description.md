# Resume Description

## Project Title

AI Statistical Reporting Workflow Demo

## One-Line Version

Built a pure Python, human-in-the-loop AI workflow demo that turns synthetic social science data and an analysis plan into ANOVA diagnostics, APA-style reports, tables, figures, Word output, and quality checks.

## Resume Bullet Options

- Designed a human-in-the-loop AI statistical reporting workflow for social science research, covering data validation, ANOVA suitability checks, human approval gates, automated analysis, APA-style reporting, tables, figures, and Word export.
- Built a pure Python MVP using pandas, scipy, statsmodels, pingouin, matplotlib, python-docx, and YAML configuration to reduce local setup friction for GitHub reviewers and interviewers.
- Converted a repetitive research reporting process into a modular workflow with explicit guardrails, reusable reporting components, and output consistency checks.
- Defined public-safe mock data, privacy audit steps, and showcase documentation to demonstrate AI workflow product thinking without exposing private research materials.

## Interview Talking Points

- The project starts from a real repeated workflow: researchers often need to move from data checks to statistical results, tables, figures, and manuscript-ready reports.
- The MVP avoids over-automation by stopping before formal conclusions and requiring human approval for the analysis path.
- The technical route is intentionally pure Python to make the demo easy to inspect and run locally.
- The repository demonstrates product thinking: scoped MVP, workflow guardrails, reusable modules, quality checks, and privacy-safe public presentation.
